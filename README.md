# Route Planner DZ

Subject: The exploitation of data from the National Addressing Referential of Algeria for route calculation

The application calculates the route between two or more points (navigation), based on a set of addressing data standardized according to the Algerian addressing standard established the technical subcommittee, Open source tools and data have been heavily used in this project, it used MongoDB as an address database and PostgreSQL for Algeriaâ€™s geographic data, Geoserver is used as a cartographic server and Leaflet as an interactive maps.
The user interface was developed with HTML / CSS and JavaScript (Vue.js & Bootstrap) and for the backend Flask (Python) was used.
